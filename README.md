# Bem-te-vi — Bar & Lanches

Site institucional do **Bem-te-vi Bar & Lanches**, localizado em Palmas, Tocantins.

## Tecnologias

- HTML5 + CSS3 puro (sem frameworks)
- Fontes: Playfair Display + Poppins (Google Fonts)

## Como rodar

```bash
npm run dev
```

Acesse em: [http://localhost:3000](http://localhost:3000)

## Estrutura

```
bem-te-vi/
├── index.html   # Todo o site (HTML + CSS + JS)
├── logo.png     # Logo do bar
├── foto.png     # Foto principal
├── wats.png     # Ícone do WhatsApp
└── package.json
```
